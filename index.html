GitHub Setup

To get started, you need a GitHub Personal Access Token (PAT) to store your data securely.

How to create a PAT:

  1. Go to GitHub Settings ‚Üí Tokens
  2. Click "Generate new token" (Fine-grained)
  3. Configure:
    • Token name: Wheel of Life Tracker
    • Expiration: 1 year
    • Repository access: Only select repositories ‚Üí hrunkaru/wheel-of-life
    • Permissions: Contents ‚Üí Read and write
  4. Click "Generate token" and copy it
This will be stored securely in your browser
